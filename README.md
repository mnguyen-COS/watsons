# Watson's Organic Market & Cafe - Website Redesign
Redesigned website of Watson's Organic Market &amp; Cafe for GD107 at COS.
## About Watson's
[Watson's](https://watsonsvisalia.com/) is a business in downtown Visalia, California that sells organic products and hosts a casual cafe with soups, sandwiches, and beverages.
## About the project
This website redesign is created by M. Nguyen at COS using HTML, CSS, and JavaScript. It aims to upgrade the existing business site for Watson's by updating its design and adding more content (webpages & photos).
