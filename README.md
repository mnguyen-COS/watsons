# Watson's Organic Market & Cafe - Website Redesign
Redesigned website of Watson's Organic Market &amp; Cafe by M. Nguyen on May 17, 2026 for GD107 at COS, located at https://mnguyen-cos.github.io/watsons/.
## About Watson's
[Watson's](https://watsonsvisalia.com/) is a business in downtown Visalia, California that sells organic products and hosts a casual cafe with soups, sandwiches, and beverages.
## About the project
This website redesign is created using HTML, CSS, and JavaScript. It aims to upgrade the existing business site for Watson's by updating its design and adding more content (webpages & photos). While the original website is a single static page, this redesign contains a home page, about page, menu page, and contact page. It is responsive for desktop, mobile, and tablet.
### Fun stats
- Four pages
- Eight photos
- Two external links
- One internal link
- 29 drinks on menu
- 29 food items on menu
